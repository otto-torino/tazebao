Benvenuto!

Tazebao è una web application multi-utente che consente di gestire campagne newsletter. Tazebao implementa un web service REST che ti permette di gestire esternamente la sottoscrizione e disiscrizione degli utenti in massima sicurezza. Tazebao gestisce in maniera asincrona l'invio massivo di email utilizzando le migliori tecnologie disponibili!

Tazebao è Open Source
